Yes, this website!

I started planning this website around a month before I actually started working on it. First I intended to use Jekyll, which is a static site generator that integrates very well with GitHub Pages, which I am currently using to host this site. I could not find a template that I felt was perfect, so I just started writing and going along with the documentation.

Roadblocks I encountered

One of the major problems I encountered while doing this was everything changing when editing style.css. I try to align something to the centre and something moves where it shouldn't.

How well I think I performed

I think I did rather well, considering this is only the second time I've done any sort of programming, and I went in with no experience whatsoever, and learned during the process. The source code could do some housekeeping since it's so hard to read. This is still an ongoing project with progress being done constantly. I am currently thinking about buying a domain name for this site or lending a free .tk or .cf domain name.
